
**Calendar Client (Frontend)**
Frontend application for a full-stack calendar system built with native JavaScript.

* Provides user interface for authentication (email login and GitHub OAuth integration)
* Allows users to create, view, and manage calendar events
* Supports event participation workflows (invites, status updates, role-based actions)
* Displays multiple calendars simultaneously with shared visibility support
* Communicates with backend REST API for all data operations

**Tech Stack:** Vanilla JavaScript, HTML, CSS

Backend API: https://github.com/leonShapiro/calendar-server
